# Sample Data

Sample NetCDF files for trying GeoScope. Generated by [spmodel-rs](https://github.com/daktu32/spmodel-rs) and subsampled for smaller file sizes.

## Files

| File | Size | Description | Features demonstrated |
|------|------|-------------|----------------------|
| `rossby_haurwitz_sample.nc` | 1.3 MB | Rossby-Haurwitz wave (Williamson test case 6) | Globe, Map, contour, vector overlay, streamlines |
| `beta_gyre_sample.nc` | 8.1 MB | Beta-plane gyre with trajectory tracking | Trajectory overlay, time animation |
| `held_suarez_sample.nc` | 5.3 MB | Held-Suarez (1994) atmospheric benchmark, 10 sigma levels | Cross-section, level selection, profile |

## Quick Start

```bash
# From the geoscope directory:
cargo run --release -- samples/rossby_haurwitz_sample.nc
```

## Details

### rossby_haurwitz_sample.nc
- Variables: `vort` (vorticity), `phi` (geopotential), `u_cos`, `v_cos` (wind)
- Grid: 128 x 64 (lon x lat), 21 time steps
- Try: Globe view + contour overlay (C) + streamlines (V)

### beta_gyre_sample.nc
- Variables: `vort` (vorticity), `u_cos`, `v_cos` (wind), `gyre_lon`, `gyre_lat` (trajectory)
- Grid: 128 x 64 (lon x lat), 51 time steps
- Try: Globe view + trajectory overlay (T) + time animation (Space)

### held_suarez_sample.nc
- Variables: `vort` (vorticity), `temp` (temperature), `ucos`, `vcos` (wind), `lnps` (log surface pressure)
- Grid: 64 x 32 x 10 levels (lon x lat x lev), 21 time steps
- Try: Cross-section view (6) + level slider + profile view (5)

## Full Data

Full-resolution data can be generated with spmodel-rs:

```bash
cd ../spmodel-rs
cargo run --release --example rossby_haurwitz   # → output/rossby_haurwitz.nc (62 MB)
cargo run --release --example beta_gyre         # → output/beta_gyre.nc (158 MB)
cargo run --release --example held_suarez       # → output/held_suarez.nc (26 MB)
```
